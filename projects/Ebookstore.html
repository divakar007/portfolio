E-commerce Website for a Bookstore

Description: This website is an E-commerce platform designed for a bookstore, offering users the capability to browse and add books to their cart. Developed using React JS with Java powering the backend, the application provides users with the flexibility to choose books from various categories. Users can effortlessly add books to their cart and seamlessly proceed to the checkout process. Thorough checks and validations have been implemented to ensure a smooth and secure user experience.
