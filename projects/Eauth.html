E-Authentication System Using OTP and QR code

It is a two-factor authentication system for websites. In the first step, the user is authenticated using his/her email and password. In the second step, the user has to choose a mode of authentication user can choose either OTP or QR code authentication. QR code / OTP will be sent to the user's email and the user need to submit the OTP/QR code. after successful submission user will be logged into his/her account.
